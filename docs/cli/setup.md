# `promptx setup`

Initialize `~/.promptx/promptx.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
promptx setup
promptx setup --workspace ~/.promptx/workspace
```

To run the wizard via setup:

```bash
promptx setup --wizard
```
